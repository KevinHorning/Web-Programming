QUERY TO CREATE PRODUCT TABLE:

CREATE TABLE Products( ProductID INT auto_increment NOT NULL, ProductName VARCHAR(40) NOT NULL, SupplierID INT(11), CategoryID INT(11), QuantityPerUnit VARCHAR(20), UnitPrice DECIMAL(10,4), UnitsInStock SMALLINT(2), UnitsOnOrder SMALLINT(2), ReorderLevel SMALLINT(2), Discontinued BIT(1), PRIMARY KEY (ProductID) );

Query to get Product list (id, name, unit price) where current
products cost less than $20:

SELECT ProductID, ProductName, UnitPrice FROM Products WHERE UnitPrice < 20.0;

Product list < $20 Query Screenshot:


Query to get Product list (id, name, unit price) where products cost between $15 and $25:

SELECT ProductID, ProductName, UnitPrice FROM Products WHERE UnitPrice < 25.0 AND UnitPrice > 15.0;

Product list $15-$25 Query Screenshot:


Query to get Product list (name, unit price) of above average price:

SELECT ProductName, UnitPrice FROM Products WHERE UnitPrice > (SELECT AVG(UnitPrice) FROM Products);

Product list > average price Query Screenshot:


Query to get Product list (name, unit price) of ten most expensive products.

SELECT ProductName, UnitPrice FROM Products ORDER BY UnitPrice desc LIMIT 10;

Query to get count current and discontinued products:

SELECT COUNT(ProductID) AS Discontinued FROM Products Where Discontinued = TRUE;

SELECT COUNT(ProductID) AS Active FROM Products Where Discontinued = FALSE;
