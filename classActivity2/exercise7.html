Design Process:
  • Style hierarchy will be implemented, with the school having default settings (background, font, font size, font color, table style etc.) across all web pages on the school website
  • If teachers want to modify the style on their specific web page, they can do so
  • This will be done by having all web pages linked to external style sheets that specify all the default school website style settings
  • When teachers want to modify their page, they must use either tag attributes or an embedded style sheet
Useful HTML tags and attributes for website:
  • <p>
    • font size
    • font family
    • color
  • <img>
    • height
    • weight
  • <a href>
  • <table>
    • font size
    • font family
    • color
  • <hr />
    • color
    • width
  • <br />
    • width
  • <ol>
    • numbering type
  • <ul>
    • bullet type
  • <h1>
    • font size
    • font family
    • color
  • <form>
    • margins
    • alignment
